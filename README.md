# cis3020project
Paycheck to Paycheck
Learn budgeting without the spreadsheets - An interactive financial literacy game

About
Paycheck to Paycheck is a browser-based game that makes learning budgeting fun. Experience realistic financial decisions, see immediate consequences, and build money management skills—all without risking real cash.
Why This Game?

Traditional budgeting apps are boring - We made it fun
Financial education is abstract - We made it experiential
Real mistakes are expensive - Practice safely here

How to Play

Choose difficulty - Pick your financial situation
Make decisions - Pay, delay, or skip expenses
Manage 3 stats - Balance, credit score, stress level
Complete 12 months - Survive a full year
Get scored - See how well you budgeted


Tech Stack

HTML5, CSS3, Vanilla JavaScript
Web Audio API (sound effects)
LocalStorage (save data)
